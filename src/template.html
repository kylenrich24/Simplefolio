About me

I am a Web Application Developer who uses the popular MERN (Mongoose, Express, React and Node) Stack.

DevConnector

DevConnector is a social network for developers. Users of this app can register and sign in, build their profile/portfolio, view other developers' profiles, make posts, comments and post likes and dislikes.This web app consists an extensively built backend API that is created via NodeJS and Express and tested thoroughly with Postman. React and Redux were used to create the frontend to work elegantly with the API and this is also tested using Redux DevTools. Lastly, the app is deployed to Heroku using Git.

Emaily

Emaily is a live web application that can be used to send mass emails to a big list of users for the purpose of collecting feedback for a product. ES10 is used to develop a RESTful API Server in Express and resources in React and Redux. MongoDB is integrated for a live database and GoogleOAuth for authentication of users. Handing of payment is handled via Stripe APIs. Lastly, transactional emailing and feedback collection is handled via SendGrid API
